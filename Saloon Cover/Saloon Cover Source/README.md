# Saloon Cover

Adds the vanilla bouncer inside the Stardrop Saloon near the right side of the entrance. On configured days after the configured start time, entering the Saloon asks whether you want to pay the cover charge.

## Features

- Uses the vanilla `Characters/Bouncer` texture at runtime.
- Draws the bouncer inside the Saloon near the right side of the entrance.
- Shows a pay-or-leave prompt when entering after the configured start time.
- Configurable cover charge price.
- Configurable cover start time.
- Configurable active days of the week.
- Sends the player back outside if they refuse or cannot afford the cover.
- Optional Generic Mod Config Menu support for cover amount, start time, active days, pay-once-per-day behavior, bouncer tile placement, and debug logging.

## Install

1. Install SMAPI.
2. Put the built `Saloon Cover` folder in your `Mods` folder.
3. Launch the game through SMAPI.
