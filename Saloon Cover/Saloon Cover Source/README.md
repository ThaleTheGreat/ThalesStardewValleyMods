# Saloon Cover

Adds the vanilla bouncer inside the Stardrop Saloon near the right side of the entrance. After 8pm, entering the Saloon asks whether you want to pay a 15g cover charge by default.

## Features

- Uses the vanilla `Characters/Bouncer` texture at runtime.
- Draws the bouncer inside the Saloon near the right side of the entrance.
- Shows a pay-or-leave prompt when entering after 8pm.
- Charges 15g by default if the player chooses to pay.
- Sends the player back outside if they refuse or cannot afford the cover.
- Optional Generic Mod Config Menu support for cover amount, start time, pay-once-per-day behavior, bouncer tile placement, and debug logging.

## Install

1. Install SMAPI.
2. Put the built `Saloon Cover` folder in your `Mods` folder.
3. Launch the game through SMAPI.
